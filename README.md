# WPortfolio

This is my personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**.  
It showcases my projects, skills, and experiences as a web developer.

## Features

- Responsive design for desktop and mobile
- Projects section with images and links
- About and Contact sections
- Smooth animations and interactive UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Git & GitHub

## Setup

To run the project locally:

```bash
# Clone the repo
git clone https://github.com/Wiamfrikal05/WPortfolio.git
cd WPortfolio-master

# Install dependencies
npm install

# Start the development server
npm run dev
